CLI Reference¶

Please find an examples of CLI Tool usage here.

See full list of CLI options below.

Primary command:

spoonbill¶

CLI tool to flatten OCDS datasets

spoonbill [OPTIONS] FILENAME

Options

--schema <schema>¶

Schema file uri

--selection <selection>¶

List of tables to extract

--split <split>¶

List of tables to split into multiple sheets

--threshold <threshold>¶

Maximum number of elements in array before its spitted into table

--state-file <state_file>¶

Uri to previously generated state file

--xlsx <xlsx>¶

Path to result xlsx file

--csv <csv>¶

Path to directory for output csv files

--combine <combine>¶

Combine same objects to single table

--unnest <unnest>¶

Extract columns form child tables to parent table

--unnest-file <unnest_file>¶

Same as –unnest, but read columns from a file

--only <only>¶

Specify which fields to output

--only-file <only_file>¶

Same as –only, but read columns from a file

--repeat <repeat>¶

Repeat a column from a parent sheet onto child tables

--repeat-file <repeat_file>¶

Same as –repeat, but read columns from a file

--count¶

For each array field, add a count column to the parent table

--human¶

Use the schema’s title properties for column headings

--language <language>¶

Language for headings

Options

en | es

-v, --verbosity <LVL>¶

Either CRITICAL, ERROR, WARNING, INFO or DEBUG.

Arguments

FILENAME¶

Required argument
